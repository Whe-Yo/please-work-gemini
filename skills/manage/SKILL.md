# Manage — 스킬 관리

장착된 스킬을 주기적으로 점검하고 이 저장소의 최신 버전과 동기화한다.

## 명령별 절차

### 현재 스킬 목록 확인
1. 에이전트의 스킬 폴더에 어떤 스킬이 장착되어 있는지 나열한다.
2. `skills/SKILL_INDEX.md`와 대조해 미장착 스킬과 구버전 스킬을 식별한다.
3. 결과를 보고한다.

### 스킬 추가
1. `skills/SKILL_INDEX.md`에서 추가할 스킬의 설명을 확인한다.
2. 현재 환경에서 해당 스킬이 동작 가능한지 검토한다.
3. 스킬 폴더를 에이전트 스킬 디렉토리에 복사한다.
4. 장착 완료 및 트리거 조건을 보고한다.

### 스킬 업데이트
1. 이 저장소(`Harnessing_Agent`)의 최신 변경사항을 확인한다 (`git pull` 또는 수동 확인).
2. 장착된 스킬과 저장소의 최신 버전을 diff로 비교한다.
3. 변경된 스킬만 선택적으로 갱신한다. 전체 덮어쓰기는 하지 않는다.
4. 변경 내용을 요약해 보고한다.

### 스킬 제거
1. 제거 대상 스킬 폴더를 에이전트 스킬 디렉토리에서 삭제한다.
2. 제거 이유를 RPW 문서에 기록한다.

## 규칙
- boost와 worklog는 제거 요청이 있어도 사용자에게 한 번 확인한다 (세션 연속성 핵심 스킬).
- 업데이트 시 기존 스킬에 사용자가 직접 수정한 내용이 있으면 덮어쓰기 전에 diff를 보여준다.
